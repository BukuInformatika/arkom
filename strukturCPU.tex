\section{Struktur CPU}
\subsection{Komponen CPU
CPU (Central Processing Unit) adalah komponen yang paling penting pada sistem komputer, CPU mengolah data berdasarkan instruksi yang diterimanya.
Berikut beberapa komponen CPU:
1. Arithmatic and Logic Unit (ALU) memiliki tugas membentuk fungsi pengolahan data pada komputer, atau bbisa juga disebut mesin bahasa pada komputer
2. Control Unit bertugas mengkontrol operasional CPU sehingga tercipta singkronisasi antara komponen dalam bekerja.
3. Registers sebagai media penyimpanan internal pada CPU pada saat pengolahan data .
4. CPU Interconnections berfungsi menghubungkan semua komponen, yaitu ALU, Unit Control, Registers, beserta dengan komponen yang berkaitan dengan sistem lainnya.
